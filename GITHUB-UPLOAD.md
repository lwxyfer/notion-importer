# GitHub Upload 1.0.1

Ce dossier est prêt pour l'upload du repo source `FEUAZUR/notion-importer`.

## Contenu inclus

- source du plugin
- assets
- tests
- `package.zip` de release
- version `1.0.1` dans `plugin.json` et `package.json`

## Ce dossier n'inclut pas

- `node_modules`
- `dist`
- `tests-dist`
- `tmp`

## Sur GitHub

1. Ouvrir le repo `FEUAZUR/notion-importer`
2. `Add file` -> `Upload files`
3. Glisser tout le contenu de ce dossier dans GitHub
4. Commit sur la branche voulue
5. Aller dans `Releases`
6. Créer la release `v1.0.1`
7. Joindre `package.zip`
